Garbage Classification Using Convolution Neural Network (CNN)

As the population is increasing, the generation of waste is also rising at a high rate.The accumulation of waste has become a major threat to the environment and life especially in the urban regions. This highlights an urgent need for an advanced and intelligent waste management system that can help in segregating and disposing of waste materials in such a way that it doesnâ€™t harm our environment.


A part of waste management involves separation of different waste which is usually done manually by hand-picking. This task can be done faster and efficiently by using a Convolution Neural Network(CNN) model that can differentiate between different kind of waste products. This model is trained on the dataset a Garbage Classification dataset extracted from Kaggle which contains 6 different types of waste materials-Cardboard, Glass, Metal, Paper, Plastic and Trash.

This model is built using CNN which are known for image classification and recognition with high accuracy. When feeded with an image of a garbage item as an input, this model is able to classify its type out of the 6 categories mentioned above with 96.79% accuracy.
